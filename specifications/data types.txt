# Data Types

## Signed Integer

    Int         Platform dependend signed integer (1, 2, 4 or 8 Byte)
    Int8        1 Byte signed integer
    Int16       2 Byte signed integer
    Int32       4 Byte signed integer
    Int64       8 Byte signed integer

## Unsigned Integer

    UInt        Platform dependend unsigned integer (1, 2, 4 or 8 Byte)
    UInt8       1 Byte unsigned integer
    UInt16      2 Byte unsigned integer
    UInt32      4 Byte unsigned integer
    UInt64      8 Byte unsigned integer

## Floating Point

    Float       Platform dependend floating point number (2, 4 or 8 Byte)
    (Float16)   2 Byte floating point number
    Float32     4 Byte floating point number
    Float64     8 Byte floating point number

## Boolean

    Bool        1 Byte truth value
    (Bool16)    2 Byte truth value
    (Bool32)    4 Byte truth value
    (Bool64)    8 Byte truth value

## String

    String      Character sequence with variable length
